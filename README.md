# 🎨 My Portfolio

A **personal portfolio website** built using **React and Bootstrap**.  
This portfolio showcases my skills, projects, and contact information in a clean and modern design.

---

# 🚀 Features

## 🏠 Home Page
* 👋 Introduction and profile picture  
* 🎯 Overview of skills 
* 🌐 Links to social profiles  

## 💼 Projects Section
* 📌 Showcase of completed projects  
* 🔗 Live demo links for each project  
* 📝 Short description for each project  

## 📬 Contact Form
* 💌 Send messages directly to email  
 

---

# 📱 Responsive Design
The portfolio is fully **responsive** and works smoothly on all devices:  
* 💻 Desktop  
* 📱 Mobile  
* 📟 Tablet  

UI is implemented using **Bootstrap Grid System and React Components**.
